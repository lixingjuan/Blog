# typeof value

1. 只能判断基本数据类型 (undefined, null, )
2. 除了Function,其他的使用能用new操作符创建的都是Object
3. 变量未声明返回'undefined'而不会抛出错误


```javascript
/* 一些特殊例子 */
typeof a // object
typeof b // undefined
typeof c // object 
typeof d // number
```
